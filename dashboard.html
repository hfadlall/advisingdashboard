Past Courses Report

  1. Go to Stellic → Reports → Student Report
  2. Select all students (or filter to relevant students)
  3. Under Student Basic, keep all 5 options selected
  4. Under Demographics, keep all 4 options selected
  5. Under Plan, select All Past Courses
  6. Generate and download the report

Current Courses Report

  1. Go to Stellic → Reports → Student Report
  2. Select all students (or filter to relevant students)
  3. Under Student Basic, keep all 5 options selected
  4. Under Demographics, keep all 4 options selected
  5. Under Plan, select Registered Courses for Specific Semester
  6. Choose the current semester (e.g., Spring 2026)
  7. Check the "Official" box
  8. Generate and download the report
